# 👟 ShoeVerse - Modern E-Commerce Shoe Store

![Project Banner](https://github.com/Kalyan-github-4/ShoeVerse/blob/main/ShoeVerse.png?raw=true)

A frontend-only React e-commerce UI built with Vite for blazing-fast performance. Features advanced product filtering, smooth scrolling, and responsive design.

## ✨ Features

### 🛒 Core Functionality

| Feature               | Description                                 | Tech Used                 |
|-----------------------|---------------------------------------------|---------------------------|
| **Advanced Filters**  | Filter products by multiple criteria        | React Context API         |
| • Category Selection  | Choose from Sneakers, Flats, Sandals, Heels| Controlled Radio Buttons  |
| • Price Range         | Range-based product filtering               | Custom Slider + State     |
| • Brand Filter        | Filter by popular brands like Nike, Adidas | Filter Tags + State Logic |
| • Color Selector      | Select shoes by visible color swatches     | CSS Color Swatches        |
| **Search Bar**        | Live product search with keyword suggestions| Debounced Input + Filtering Logic |
| **Product Cards**     | Interactive cards with prices, ratings, and discounts | Reusable Components |
| **Sorting System**    | Sort shoes by rating, price, or arrival     | Array Sorting Methods     |
| **Responsive Design** | Optimized across desktop and mobile views   | Flexbox & Media Queries   |
| **Wishlist & Cart**   | Icon-based quick access to wishlist/cart    | React Icons + State Hooks |
| **Animated Scrolling**| Smooth scroll transitions across product sections     | `scroll-behavior: smooth` + JS |



### 🎨 UI/UX Highlights

- 🧭 Easy navigation with fixed sidebar and tabs  
- 💡 Animated product transitions and modal previews  
- 📱 Mobile-first responsive layout  
- ♿ Accessibility-first design approach

## 🚀 Getting Started

### ✅ Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended) — only required to run the Vite dev server, not for backend
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) — to manage project dependencies
